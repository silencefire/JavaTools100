#工程描述
用于记录常用的工具类，包括：JDK自带工具类，以及第三方库工具类；主要用于存储java开发中遇到的各种工具类，存储各种“轮子”
##jdk工具类
 - Objects工具类（jdk11引入）
 - Math工具类
 - Arrays
 - BigDecimal
 - SimpleDateFormat
 - Calendar
 - System
 - String
 - Collections集合工具类

##第三方库
 - org.apache.commons
   - lang
   - lang3
   - io
   - beanutils
 - org.apache.http.util
 - org.springframework.util
 - guava类库（提供集合，缓存，支持原语句，并发性，常见注解，字符串处理，I/O和验证的实用方法）
 - Hutool（小众，国内）
